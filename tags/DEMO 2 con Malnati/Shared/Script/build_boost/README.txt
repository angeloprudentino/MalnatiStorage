**************************************
* Istruzioni per compilare Boost
**************************************

- scaricare l'archivio con il sorgente da qui:  http://sourceforge.net/projects/boost/files/boost/1.59.0/boost_1_59_0.7z/download
- estrarre l'archivio con il comando "estrai qui" (cioè direttamente in una cartella di nome "boost_1_59_0")
- lanciare lo script build_boost.bat dalla cartella in cui si trova (opzione --help per la lista di parametri)
- alla fine della compilazione lo script indicherà quali sono i path da inserire nel progetto Visual Studio
  per poter usare la libreria
